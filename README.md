nuggets-miner
=============

Nuggets Miner, a recommendation system in the form of a plugin for the Eclipse IDE to assist software developers in development and maintenance tasks.

Paper URL: http://lascam.facom.ufu.br/cms/userfiles/downloads/2014/CbsoftToolsEduardo2014preprint.pdf

Nuggets Miner, awarded the 2nd best tool at CBSoft2014. Oct, 2014 (Maceió-Alagoas).

Authors: Eduardo Cunha Campos, Lucas Batista Leite de Souza and Marcelo de Almeida Maia

To execute Nuggets Miner, the requirements of the target system are:

- Java-based systems;
- Apache Solr configuration. Available at: http://lucene.apache.org/solr/4_10_1/tutorial.html
- Eclipse IDE (Version): Eclipse Helios SR2 Packages (v 3.6.2).

Additional information about Nuggets Miner:

- Operating system(s): MacOS, Linux, and Windows.
- Programming language: Java.
- Nuggets Miner is an open source project, distributed under a MIT license.
